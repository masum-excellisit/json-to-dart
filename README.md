# json-to-dart

Api json response to dart model

# How to use

Link : https://masum-excellisit.github.io/json-to-dart/

1. Go to the site
2. Paste your json response in the left side box
3. Click on the "Convert to Dart" button
4. Your dart model will be generated in the right side box
5. Copy the generated dart model and use it in your project
